# Object Detection Recognition System
📌 Project Overview
This project is a real-time **Object Detection  Recognition System** developed using **Python and YOLOv5**.
It detects objects from live camera input and also performs face recognition.

The system is useful for applications such as:
* Smart surveillance
* Security systems
* Automated monitoring
* AI-based detection systems
---

## 🛠️ Technologies Used

* Python
* YOLOv5
* OpenCV
* HTML & CSS
* Deep Learning

---

## 📂 Project Structure

Object_Detection
│
├── face.py → Face detection module
├── object.py → Object detection module
├── index.html → Web interface
├── hi.css → Styling file
├── hi.pu.py → Supporting Python script
├── yolov5s.pt → Pretrained YOLOv5 model
└── launch.json → Configuration file

---

## ⚙️ Features

* Real-time object detection
* Face detection capability
* Uses pretrained YOLOv5 model
* Simple web interface for display
* Fast and efficient detection

---

## ▶️ How to Run the Project

1. Install Python and required libraries
2. Install dependencies:

```
pip install opencv-python torch torchvision
```

3. Run the object detection script:

```
python object.py
```

4. Run the face detection script:

```
python face.py
```
